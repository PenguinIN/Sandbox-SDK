Simulation

To make our test and integration easier we have some pre-configured simulation scenarios that we use to demonstrate the functionalities of our SDK on the Sample App


nav_single_demo


In this scenario we have the path points starting from 1st floor and ending at the same floor as seen in the following image , you can navigate to 1059 POI to show Single floor Navigation.

nav_multi_demo

In this scenario we have the path points starting from 3rd floor to 2nd floor to 1st floor as seen in the following images , you can navigate to 1059 POI to show Multi floor Navigation.

error_single_floor

In this scenario we have the path points starting from 1st floor and ending at the same floor as seen in the following image , you can navigate to 1059 POI to show the error path at single floor.

error_multi_demo

In this scenario we have the path points starting from 2nd floor to 3rd floor as seen in the following images , you can navigate to 1059 POI at 1st floor to show the error path at multi floor.
